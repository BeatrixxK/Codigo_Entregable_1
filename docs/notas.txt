9. Programar la navegación en MainForm

Ahora vamos a conectar los botones para abrir los formularios.

Abre MainForm.cs y usa este código:

using System;
using System.Windows.Forms;

namespace DragonNutrex.Views
{
    public partial class MainForm : Form
    {
        public MainForm()
        {
            InitializeComponent();
        }

        private void btnUsuarios_Click(object sender, EventArgs e)
        {
            RegistroUsuarioForm form = new RegistroUsuarioForm();
            form.ShowDialog();
        }

        private void btnProductos_Click(object sender, EventArgs e)
        {
            ProductoForm form = new ProductoForm();
            form.ShowDialog();
        }

        private void btnMenus_Click(object sender, EventArgs e)
        {
            MenuForm form = new MenuForm();
            form.ShowDialog();
        }

        private void btnNutricion_Click(object sender, EventArgs e)
        {
            NutricionForm form = new NutricionForm();
            form.ShowDialog();
        }

        private void btnEstadisticas_Click(object sender, EventArgs e)
        {
            EstadisticasForm form = new EstadisticasForm();
            form.ShowDialog();
        }
    }
}
10. Conectar los eventos Click

Para que esos métodos funcionen, los botones deben tener enlazado su evento Click.

Forma más fácil

En el diseñador de MainForm:

haz doble clic sobre btnUsuarios

Visual Studio crea automáticamente btnUsuarios_Click

Haz lo mismo con:

btnProductos

btnMenus

btnNutricion

btnEstadisticas

Después reemplazas el contenido por el código que te puse arriba.

15. Errores comunes que te pueden salir
1. The name 'InitializeComponent' does not exist

Eso pasa si el form no fue creado como Windows Form sino como clase normal.
Solución: vuelve a crearlo con Add > Windows Form.

2. No encuentra el form en MainForm

Revisa que todos estén en el namespace correcto:

namespace DragonNutrex.Views
3. El botón no hace nada

El evento Click no está conectado.
Haz doble clic en el botón o revisa en propiedades el ícono de rayo.

4. Program.cs no abre MainForm

Revisa que diga:

Application.Run(new Views.MainForm());

o que arriba tengas el using DragonNutrex.Views; y pongas:

Application.Run(new MainForm());
